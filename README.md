# CrappyMagic

## API Spec

```
GET /item/:number
  returns json formatted
{
  title <String>,
  owner_name <String>,
  owner_email <String>,
  video_creator_name <String>,
  video_creator_web_address <String>,
  video_creator_email <String>,
  video_creator_instagram_handle <String>,
  price <Float>,
  description <String>,
  materials <String>,
  manufacture_info <String>,
  mature_content <Boolean>
}
```
